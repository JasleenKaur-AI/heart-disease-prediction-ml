# Heart Disease Prediction using Machine Learning

## Project Overview
This project uses Machine Learning algorithms to predict the likelihood of heart disease using healthcare datasets and predictive analytics techniques.

The project focuses on data preprocessing, Exploratory Data Analysis (EDA), feature engineering, model comparison, and predictive analytics.

---

## Business Problems
Early prediction of heart disease can help healthcare institutions reduce risk, improve patient outcomes, and support preventive healthcare strategies through data-driven insights.

---

## Features
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Correlation Heatmap
- Feature Engineering
- Model Comparison
- Hyperparameter Tuning using GridSearchCV
- Model Evaluation & Visualization

---

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

---

## Machine Learning Models Used
- Decision Tree Classifier
- Logistic Regression
- Random Forest Classifier

---

## Results
- Achieved approximately 85% prediction accuracy
- Random Forest achieved highest performance
- Improved model performance using hyperparameter tuning
